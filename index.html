Static Site Generators

Hjemmesider kan opdeles i to kategorier: Statiske og dynamiske. Et statisk websted er designet, gemt som filer på en server og leveret til brugerens webbrowser som sitet er.
Eleventy

Eleventy (11ty) is a Node.js based static site generator (SSG).
Node.js

Node.js er en server-side-platform bygget på Google Chrome's JavaScript Engine (V8 Engine).
Markdown

Markdown er et let markup-sprog, som bruges til at tilføje formatering til almindelige tekstdokumenter.
Visual Studio Code

Visual Studio Code er en freeware kode-editor lavet af Microsoft til Windows, Linux og macOS.
